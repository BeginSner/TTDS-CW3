Input Format
1. Query Format
(1) Word: No Quotation Mark, e.g. computer
(2) Phrase: Need Quotation Mark, e.g. "computer science"
2. Search Format
(1) Fuzzy Search: No Bracket, e.g. computer, "computer science"
(2) Exact Search: Need Brackets, e.g. [computer], ["computer science"]
(3) Proximity Search: [#15(Term1, Term2)], No Bracket means fuzzy search
(4) Boolean Search: e.g. [computer AND english], ["computer science" OR teacher], [[#20(Term1, Term2)] AND NOT "computer science"]
Boolean Search can be combined arbitrarily (word, phrase, proximity)
